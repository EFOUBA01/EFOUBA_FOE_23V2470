# Application de Cartographie de Santé - Yaoundé

Cette application permet de visualiser et d'analyser les données de santé pour la ville de Yaoundé.

## 🚀 Lien de l'application
Lien direct vers l'application : [https://efouba-foe-23v2470.onrender.com](https://efouba-foe-23v2470.onrender.com)

## 🛠️ Installation locale
1. Cloner le dépôt : `git clone https://github.com/EFOUBA01/EFOUBA_FOE_23V2470.git`
2. Installer les dépendances : `pip install -r requirements.txt`
3. Lancer l'interface : `streamlit run app.py`
